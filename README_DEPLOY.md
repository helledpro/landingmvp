# Пульс — лендинг

## Как открыть локально
Откройте `index.html` в браузере двойным кликом или через локальный сервер:

```bash
php -S localhost:8000
```

После запуска перейдите на `http://localhost:8000`.

> Если открыть сайт через `file://`, страница отобразится, но форма не сможет отправлять заявки: `send.php` выполняется только на сервере с PHP.

## Как загрузить на хостинг
1. Откройте файловый менеджер хостинга или подключитесь по FTP/SFTP.
2. Загрузите все файлы проекта в папку `public_html`, `www` или другую корневую папку сайта:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `send.php`
   - `config.php`
   - `privacy.html`
   - `README_DEPLOY.md` (необязательно)
   - `assets/` (если появится)
3. Убедитесь, что на тарифе включён PHP и функция `mail()`.

## Где поменять email
Откройте `config.php` и замените значения:

```php
'recipient_email' => 'owner@example.com', // куда приходят заявки
'sender_email' => 'no-reply@example.com', // от кого отправляется письмо
'site_name' => 'Пульс',
```

Для лучшей доставляемости используйте `sender_email` на домене сайта, например `no-reply@your-domain.ru`.

## Если письма не приходят
- Проверьте папку «Спам».
- Убедитесь, что в `config.php` указан правильный `recipient_email`.
- Уточните у хостинга, включена ли функция PHP `mail()`.
- Настройте SPF/DKIM для домена отправителя.
- Если хостинг блокирует `mail()`, подключите SMTP-отправку через почтовый сервис или попросите поддержку хостинга включить отправку.

## Форма заявки
Форма отправляет AJAX-запрос на `send.php`. Обязательные условия: имя, согласие с политикой, а также телефон или email. При успешной отправке пользователь увидит сообщение: «Заявка отправлена. Мы свяжемся с вами в течение 15 минут.»
